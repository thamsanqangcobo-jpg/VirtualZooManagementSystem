Virtual Zoo Management System

Overview:
The Virtual Zoo Management System is a C# console application designed to simulate the operations of a zoo.
It allows users to manage different animals, track their feeding schedules, and demonstrate object-oriented programming (OOP) concepts such as inheritance, polymorphism, method overloading, and interfaces.
This project was created as part of an OOP formative assessment.

Features:

Manage various types of animals such as mammals, birds, and reptiles.

Assign and track feeding schedules.

Demonstrates polymorphism where different animals have unique behaviors.

Uses method overloading for animal interactions.

Implements interfaces to share common behaviors among classes.

Object-Oriented Concepts Demonstrated:

Inheritance:
Derived animal classes inherit from a base Animal class.

Polymorphism:
Each animal overrides methods such as MakeSound() and Feed() differently.

Method Overloading:
Methods are overloaded to perform similar actions with different parameters.

Interfaces:
Interfaces define shared behaviors, such as IFeedable or IMovable.


Technologies Used:

Language: C#

Framework: .NET Console Application

IDE: Visual Studio



How to Run the Project:

Clone or download the repository.

Open the project in Visual Studio.

Build and run the program using Ctrl + F5.

Follow the console prompts to simulate zoo management tasks.
